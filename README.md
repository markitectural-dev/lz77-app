\# Алгоритм LZ77 — Веб-приложение



Веб-приложение для демонстрации работы алгоритма сжатия данных без потерь LZ77 (метод скользящего окна).



Курсовая работа по дисциплине «Алгоритмы и структуры данных», НИТУ МИСИС, группа БИВТ-24-2.



\## Возможности



\- Сжатие — кодирование произвольной строки алгоритмом LZ77 с настраиваемыми параметрами буфера поиска (W) и буфера предпросмотра (L)

\- Распаковка — декодирование последовательности троек (d, k, c) обратно в исходную строку

\- Пошаговая трассировка — визуализация состояния буферов на каждом шаге кодирования и декодирования

\- Корректная обработка самоперекрывающихся совпадений



\## Стек



\- Python 3.8+

\- Flask

\- HTML / CSS / JavaScript (без внешних библиотек)



\## Запуск



```bash

\# Клонировать репозиторий

git clone https://github.com/markitectural-dev/lz77-app.git

cd lz77-app



\# Установить зависимости

pip install -r requirements.txt



\# Запустить приложение

python app.py

```



Приложение будет доступно по адресу `http://127.0.0.1:5000`.



\## Структура проекта




\## Примеры



| Входная строка | Тройки | Комментарий |

|---|---|---|

| `aabaabaab` | `(0,0,a)(1,1,b)(3,5,b)` | Самоперекрывающееся совпадение |

| `abcdefg` | 7 троек `(0,0,x)` | Худший случай — нет повторений |

| `aaaaaaaa` | `(0,0,a)(1,6,a)` | Лучший случай — максимальное сжатие |
